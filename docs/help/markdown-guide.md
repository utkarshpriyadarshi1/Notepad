# Markdown Editing Guide

Notepad features a raw text editor with live Markdown previewing capability.

## 👁️ View Modes
Toggle between rendering styles using the toolbar buttons:
- **Write**: Focus purely on writing text using monospace formatting.
- **View**: Renders full Markdown HTML layout cleanly.
- **Split**: Divides the workspace to show side-by-side editing and previewing.

## ✍️ Writing Markdown
Use standard Markdown syntax in the editor to structure your notes:
- Use `# Header` for headings.
- Use `**bold**` or `*italic*` for styling.
- Create lists with `- item` or `1. item`.
- Use quotes with `> Blockquote text`.
- Embed code syntax using single backticks: `` `code` ``.

## 🛠️ Format Toolbar
A formatting helper bar is docked at the top of the editor. Highlight text or click any button to insert syntax:
- **Bold / Italic / Strikethrough**: Apply standard typeface styles.
- **Highlight**: Highlights text using HTML `<mark>` tags.
- **Lists**: Inserts numbered or bulleted list points.
- **Links**: Appends `[Link Text](URL)`.
