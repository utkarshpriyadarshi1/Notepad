# Markdown Editing Guide

Notepad features a fully-featured raw text editor with a Markdown renderer.

## ✍️ Writing Markdown
Use standard Markdown syntax in the editor to structure your notes:
- Use `# Header` for headings.
- Use `**bold**` or `*italic*` for styling.
- Create lists with `- item` or `1. item`.
- Use quotes with `> Blockquote text`.
- Embed code syntax using single backticks: `` `code` ``.

## 🛠️ Format Toolbar
A formatting helper bar is docked at the top of the Markdown editor. Highlight text or click any button to insert the syntax directly:
- **Bold / Italic / Strikethrough**: Basic style inputs.
- **Heading**: Inserts heading anchors.
- **Lists**: Inserts numbered or bulleted list points.
- **Links**: Appends `[Link Text](URL)`.
- **Horizontal Rule**: Inserts divider lines (`---`).
